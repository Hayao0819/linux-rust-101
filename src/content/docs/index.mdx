## この教材について

Linux カーネルに Rust コードが入り始めた。
しかし「カーネル開発の経験がない学生」が Rust for Linux を体系的に学べる教材は、まだ存在しない。

この教材は、**C でカーネルの基礎概念を理解してから、同じものを Rust で書き直す** というアプローチで、
カーネルプログラミングと Rust for Linux の両方を同時に身につけることを目指す。

## 設計原則

1. **C first, then Rust** — 各概念を C で先に教え、次に Rust 抽象を示す
2. **一次情報最優先** — `rust.docs.kernel.org`, `docs.kernel.org`, `lore.kernel.org` を参照
3. **API ステータス可視化** — 安定 / 実験的 / RFC をバッジで明示
4. **TODO スケルトン方式** — 穴埋め式の実習で能動的に学ぶ
5. **Safety Contract 中心** — `// SAFETY:` コメントと `# Invariants` を徹底
