## 凡例

- **型**: 構造体 (`struct`) や列挙型 (`enum`)
- **トレイト**: 振る舞いを定義するインターフェース (`trait`)
- **マクロ**: コンパイル時にコードを生成するマクロ (`macro`)

ドキュメントのベース URL: [`https://rust.docs.kernel.org/kernel/`](https://rust.docs.kernel.org/kernel/)

---

## prelude

`use kernel::prelude::*;` で一括インポートされるもの。日常的に使う型・トレイト・マクロが集約されている。

| 名前 | 種別 | 説明 |
|---|---|---|
| [`module!`](https://rust.docs.kernel.org/kernel/macro.module.html) | マクロ | カーネルモジュールのエントリポイントを定義する |
| [`Module`](https://rust.docs.kernel.org/kernel/trait.Module.html) | トレイト | モジュールの `init()` を実装するためのトレイト |
| [`ThisModule`](https://rust.docs.kernel.org/kernel/struct.ThisModule.html) | 型 | 現在のモジュールへの参照を保持する |
| `pr_info!`, `pr_err!`, `pr_warn!`, `pr_debug!` | マクロ | カーネルログ出力（printk 相当） |
| [`Result`](https://rust.docs.kernel.org/kernel/error/type.Result.html) | 型エイリアス | `core::result::Result<T, Error>` のエイリアス |
| [`Error`](https://rust.docs.kernel.org/kernel/error/struct.Error.html) | 型 | カーネルエラーコードのラッパ |

|---|---|---|

## sync — 同期プリミティブ

| 名前 | 種別 | 説明 |
|---|---|---|
| [`Mutex<T>`](https://rust.docs.kernel.org/kernel/sync/struct.Mutex.html) | 型 | スリープ可能なミューテックス。データを内包する |
| [`SpinLock<T>`](https://rust.docs.kernel.org/kernel/sync/struct.SpinLock.html) | 型 | スピンロック。割り込みコンテキストでも使用可能 |
| [`Arc<T>`](https://rust.docs.kernel.org/kernel/sync/struct.Arc.html) | 型 | アトミック参照カウントによる共有所有権 |
| [`ARef<T>`](https://rust.docs.kernel.org/kernel/types/struct.ARef.html) | 型 | C カーネルオブジェクトの参照カウントラッパ |
| [`Guard`](https://rust.docs.kernel.org/kernel/sync/struct.Guard.html) | 型 | ロックガード。スコープ終了で自動アンロック |
| [`CondVar`](https://rust.docs.kernel.org/kernel/sync/struct.CondVar.html) | 型 | 条件変数。`Mutex` と組み合わせて使用する |
| [`new_mutex!`](https://rust.docs.kernel.org/kernel/macro.new_mutex.html) | マクロ | `Mutex` の pin 初期化を行うヘルパ |
| [`new_spinlock!`](https://rust.docs.kernel.org/kernel/macro.new_spinlock.html) | マクロ | `SpinLock` の pin 初期化を行うヘルパ |
